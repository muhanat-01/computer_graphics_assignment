# Jan/April 2026 BIT4102 Practical CAT - 14-Storey Architectural Project

## Project Context
This repository contains our group practical CAT submission for **BIT4102 (Jan/April 2026, CAT)**, completed within the two-week assessment window opened on **31 January 2026** and closing on **3 March 2026**. The assignment required a complete multi-storey building design package, including architectural planning and core building services documentation.

The marking scheme (30 marks total) was structured as follows:

- (a) Architectural floor plan - 10 marks
- (b) Electrical wiring plan - 6 marks
- (c) Water and plumbing plan - 6 marks
- (d) Wi-Fi and network plan - 5 marks
- (e) Presentation and professionalism - 3 marks

A key instruction in the brief was that one typical floor could be designed and declared as representative of all upper floors. Our modelling workflow follows that rule while still showing full 14-storey elevation and massing consistency.

## Submission Objective
The central aim of this submission was to produce a coherent, realistic, and technically legible 14-storey building model in Revit, then present it with discipline-specific views and symbols that reflect professional architectural documentation standards.

## Technical Reflection
This submission demonstrates a full-stack architectural documentation workflow at student professional level: from space planning and repeatable floor strategy to service integration and presentation control. The strongest technical decisions were the use of a typical floor for efficient vertical replication, discipline-separated plans for service legibility, and the combined use of 2D, elevation, and 3D views for design validation.

The final output is therefore not just a model, but a coordinated drawing set prepared to satisfy both the technical and communication requirements of the BIT4102 practical CAT.
